# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.x   | ✔️ |
| 3.x   | ❌ |
| 2.x   | ❌ |
| 1.x   | ❌ |

## Reporting a Vulnerability

We monitor the repository with dependabot to ensure dependencies are up to date. If you notice a vulnerable dependency please raise a pull request/issue with details!
